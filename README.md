# 🤖 AI Job Trends Analysis

## 📌 Overview

This project analyzes trends in AI-related jobs using a dataset of job postings. The goal is to uncover insights such as in-demand roles, required skills, salary patterns, and hiring trends in the artificial intelligence domain.

The analysis is performed using Python in a Jupyter Notebook environment, leveraging data analysis and visualization libraries.

---

## 📂 Project Structure

```
├── Ai_Job_Analysis.ipynb   # Main notebook with data cleaning & analysis
├── ai_job_trends_dataset.csv  # Dataset used for analysis
└── README.md               # Project documentation
```

---

## ⚙️ Technologies Used

* Python 🐍
* Pandas – Data manipulation
* NumPy – Numerical computations
* Matplotlib – Data visualization
* Seaborn – Statistical visualization

---

## 🔍 Key Steps in Analysis

1. **Data Loading**

   * Import dataset using Pandas

2. **Data Cleaning**

   * Handle missing values
   * Check data consistency

3. **Exploratory Data Analysis (EDA)**

   * Understand job distribution
   * Analyze salary trends
   * Identify top skills and roles

4. **Visualization**

   * Graphs and charts to highlight insights
   * Trend analysis using visual tools
